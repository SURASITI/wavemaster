# WaveMaster - Competition Management System (Starter)

Quick start (after files added to repo):

1. Copy .env.example to .env and set your Firebase config values.
2. Install dependencies:
   npm install
3. Start dev server:
   npm run dev

Notes:
- Set up Firebase Realtime Database and rules (firebase-rules.json example).
- For production, secure writes with authentication and use transactions/cloud functions for scoring.
